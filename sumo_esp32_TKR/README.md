SUMO ESP32 | TKR

ESP32-based sumo robot developed by the TKRobotics team for competitive robotics.

Overview:

This project represents a compact and powerful sumo robot designed for real competition conditions.
The system is built around ESP32 and focuses on fast response, stability, and reliable motor control.

Features:

ESP32 microcontroller (Wi-Fi capable)

High-torque DC motors

Motor driver control system

Fast reaction for pushing and defense

Optimized code for competitive matches

Hardware:

ESP32 (38-pin)

Motor driver (L298N or аналог)

DC gear motors

Power supply module

Chassis for sumo robotics

Software:

The code implements:

motor direction control

rapid movement response

stable operation under load

competition-ready logic
